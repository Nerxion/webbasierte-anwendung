# webbasierte-anwendung
Webbasierte Anwendung, welche es Nutzern ermöglicht, Angebote für Produkte zu erstellen und Produkte anderer Nutzer zu ersteigern

## Genutzt:
- SpringBoot
- Vue3
- REST-API
 
 ## Konzepte:
 - Messaging (STOMP)
 - Validation
 - I18n
 - L10n
 - Beans
 - Dependency Injection
 - Entities / Repositiories
 - DTOs
 - Reactivity
